# 🍄 马里奥英语单词冒险游乐场 (Super Mario Words)

一款将经典红白机 **FC 超级马里奥兄弟 (Super Mario Bros.)** 与**英语单词认读教学**深度融合的创新游戏化学习系统 (Gamified Learning Solution)。

## 🌟 核心功能亮点

1. 🎮 **沉浸式金币背单词 (Core Focus)**
   - 当马里奥在关卡中顶撞砖块顶出金币，或者在空中吃到悬空金币时，金币蹦出的上方会自动缓缓浮现高对比度的**中英文双语单词学习卡片**。
   - 在快乐的跳跃闯关中，潜移默化地完成高频词汇的认读和记忆，寓教于乐。

2. 🗺️ **32个原始关卡一键穿梭 (Two-way Map Navigation)**
   - 网页顶部控制条配备了 **`🗺️ 选择关卡`** 菜单，完整集成了超级马里奥兄弟从 **World 1-1** 最终到库巴城堡决赛 **World 8-4** 的全部 32 个关卡。
   - 点选切换后瞬间传送加载，且能自动将键盘操作焦点（Focus）移回画布，玩家无需鼠标二次点击即可直接操作！
   - 支持**双向状态同步**，当马里奥在游戏内打通关卡走向水管进入下一关时，网页顶部的下拉框选项会自动实时变更同步！

3. 📺 **默认“巨屏”极清画质与抗锯齿 (Pixel-perfect Rendering)**
   - **默认巨屏启动**：游戏开机默认选中并加载最清晰震撼的“巨屏 (1440x900)”画面，整个外卡框等比例协同拉大，极大方便儿童阅读。
   - **消除模糊与撕裂线**：重构了 Unitsize 偶数强制对齐限制和 Canvas 图像渲染算法，屏蔽了浏览器二次拉伸插值，彻底消除了半像素对齐产生的条纹撕裂和抗锯齿模糊，画质透亮如水晶。

4. 📁 **PDF 文章单词一键提取后台 (Admin Word Extractor)**
   - 内置强大的词库提取后台（[admin.html](admin.html)），玩家可将任何英语阅读文章、英文绘本甚至 PDF 格式文档直接拖入浏览器。
   - 系统自动对文章进行分词、去重、过滤，并自动提取其词义、词性。一键点击后直接保存为游戏内运行的专属词库。
   - 后台配备“🏠 返回首页”按钮，打通了“提取词库 -> 开始游戏”的极简闭环。

5. 🔊 **原声音效复活与排键纠偏**
   - 完美复活了原版经典的吃金币、踩怪物、死亡等效果音以及经典的背景音乐。
   - 彻底修复了 P 键暂停后因按键冲突导致无法继续恢复的 Bug，操作极为流畅。

6. 🎨 **Kenneth 专属游戏彩蛋**
   - 游戏加载及重玩开始画面的橙底巨大像素大招牌，已定制改写为了全大写圆润像素体的 **`KENNETH SUPER MARIO`** 专属彩蛋招牌，彰显版权与个性创意！

## 🛠️ 技术栈与依赖说明

- **核心语言**：原生 HTML5 Canvas 渲染、Vanilla JavaScript 纯客户端逻辑、CSS3 磨砂玻璃（Glassmorphism）响应式布局。
- **第三方库**：`PDF.js`（用于后台 PDF 纯前端文本提取与分词）。
- **静态托管**：`Vercel` 生产级一键推送发布与高性能 CDN 托管。

- 下载本代码包后，**直接双击主目录下的 `index.html` 即可在本地浏览器中完美运行游玩！**
- 后台可直接双击 `admin.html` 打开。
